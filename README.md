# 🚀 Hababah Studio Engine v1.0

Welcome to the core automated content runtime engine for **Hababah Studio**. This tool automatically drafts high-retention scripts, renders natural AI voiceovers, and fetches matching stock footage directly on your mobile device.

---

## 🛠️ Quick Mobile Setup (Termux)

Follow these two simple steps to deploy the studio engine on your Android device:

### Step 1: Install Termux
Download and install the official Termux app via F-Droid:
👉 [Download Termux here](https://f-droid.org/packages/com.termux/) *(Do not use the Google Play Store version as it is outdated).*

### Step 2: Run the One-Click Installer
Open your newly installed Termux app, copy the entire command block below, paste it inside the terminal, and press **Enter**:

```bash
pkg update -y && pkg install python python-cryptography ffmpeg -y && pip install requests edge-tts && mkdir -p ~/storage/downloads/Hababah_Studio && curl -L -o Hababah_Studio_Engine.pyc [https://raw.githubusercontent.com/Hababah/Hababah-Studio-Engine/main/Hababah_Studio_Engine.pyc](https://raw.githubusercontent.com/Hababah/Hababah-Studio-Engine/main/Hababah_Studio_Engine.pyc) && python Hababah_Studio_Engine.pyc


---

## 🔒 Activation & Licensing

Once the installation finishes, the engine will initialize and temporarily lock, displaying your unique **Machine ID**. 

To activate your software lifetime license:
1. Copy the **Machine ID** shown on your Termux screen.
2. Click here to [Message Support on WhatsApp](https://wa.me/2349138241277?text=Hi%20Hababah%20Studio,%20here%20is%20my%20Machine%20ID:%20) to instantly send your ID and verify your access.
3. Place the provided `license.key` file into your device's `~/storage/downloads/Hababah_Studio/` folder.
4. Run `python Hababah_Studio_Engine.pyc` again to unlock the full power of Hababah Studio!
